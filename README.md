# 5-java-nk-gomcha-tycoon

## 프로젝트명 - 곰차 타이쿤(GOMCHA TYCOON)

### 사용한 언어
  - Java  
### 프로젝트 설명
  - 곰차 타이쿤은 곰차를 방문한 손님으로부터 주문을 받아, 주문의 내용 및 순서에 맞게 버튼을 누르고  
총 가격까지 입력하여 주문 미션을 완료하는 시나리오로 구성되어 있습니다.
